🪖 中国艦艇の甲板 — 作戦指令のやり取り

夜明け前、東シナ海の波は黒く揺れ、台湾近海の霧が艦隊を覆っていた。
甲板では、中国海軍の指揮艦「龍雲号」の通信室が慌ただしい。緊急通信が次々と飛び交う。

⸻

指揮官と兵士のやり取り

指揮官（艦長）：「各艦、標的海域への進入を確認。火器管制は待機。緊急時のみ使用。理解したか？」

兵士A（通信班）：「はい、艦長。すべての砲塔・ミサイルは待機状態です。」

兵士B（レーダー監視）：「艦長、台湾軍の艦艇が接近中。距離5.2海里、速度20ノット。」

艦長：「攻撃は厳禁。交戦は指示待ち。衝突回避ルートを確保せよ。」

兵士C（機関室）：「エンジン出力上げます。回避行動可能です。」

⸻

緊張の瞬間

霧の向こうに台湾軍の哨戒艇が姿を現す。甲板上の兵士たちは息をひそめ、艦長の指示を待つ。

艦長：「すべての艦、光学・赤外線で監視せよ。民間船確認。誤射は絶対に許さない。」

兵士A：「了解。民間船、港近くに1隻確認。」

艦長：「航路を迂回させろ。避難時間を稼ぐんだ。」

⸻

心理描写

兵士Bは心中でつぶやく。

「本当にこれでいいのか…。戦闘を開始すれば、民間人に被害が出る。だが命令は絶対…。」

艦長もまた葛藤していた。命令通りに進めば作戦は完了するが、民間被害は避けられない。彼の手はわずかに震え、通信端末のボタンを握りしめる。

⸻

作戦指示の続行

艦長：「全艦、航行速度を維持しつつ、警告信号を発信。接触は回避、誤射なし。」

艦隊は霧の中で慎重に進み、台湾軍の哨戒艇とは一定距離を保つ。緊迫の甲板には、命令と良心の狭間で葛藤する兵士たちの静かな緊張が漂う。
