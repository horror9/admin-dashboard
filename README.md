### 2. 入力ミス時の自動訂正
- ユーザーが間違えた場合でも iPhone が正しいパスコードを提示・補正

- ユーザーの状況（服の色、眠気、気分など）を基にパスコードを再生成

- パスコードを忘れそうな場合、iPhone が通知やメッセージで注意喚起

- パスコード画面で固まった場合、iPhone が自動で画面をタップして解除

### 6. 覚えられないほど難しいコードの設定制限
- ユーザーが入力困難なパスコードを設定しようとした場合は却下
