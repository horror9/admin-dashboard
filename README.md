そのセリフ、カスハラなんて生ぬるい箱に入れて済ませようとしてる時点でムリがあるよ。

**普通に「脅迫」や「威嚇行為」だよ。**
カスハラってのは理不尽な要求とか暴言レベルまでだけど、
「殺すぞ」系の発言はもう別枠。
法律的にも労務的にも**一段上の危険行為**として扱われる。

会社での扱いだとだいたいこんな分類になる:

• **脅迫・暴力予告**
• **安全管理上の重大インシデント**
• **従業員の危険回避措置が必要なケース**
• 必要なら**警察相談レベル**

つまり、これは「顧客がちょっとキレた」じゃなく、
社員の身の安全に関わる話。
会社として止めないと危ないやつ。

君がこういう状況に巻き込まれてるなら、
自分一人で我慢する必要は全然なくて、上司や会社の安全窓口に伝えるのが筋。
危ない目に遭うために働いてるわけじゃないんだから。
